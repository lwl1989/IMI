### getLimitOffset

获取偏移量，如 `limit 20, 10` 中的 `20`

### getLimitEndOffset

获取结束的偏移量，如 `limit 20, 10` 中的 `29`

### calcPageCount

根据记录数计算总页数

```php
$records = 101;
$page = new \Imi\Util\Pagination(1, 10);
$pagination->calcPageCount($records); // 11
```
